# Project for Bilibili

处理 Bilibili Android App 端下载的媒体文件

<hr>

- Main: pivot

- 执行合成有声版方法会调用 FFMPEG (需要早已安装配置), 执行合成无声版方法可以直接执行

- input_path: 原始媒体文件所在目录

- output_path: 输出文件的目录

<hr>

# Attention

运行项目时，必须进入到 Index 所在的当前目录，执行`python $[Index].py`，而不是执行`python src/$[Index].py`
